A Gutenberg block that renders classic WordPress navigation menus via wp_nav_menu().

== Description ==

Legacy Nav Menu Block lets you add classic WordPress navigation menus to the block editor. It supports menu selection, theme locations, container configuration, depth control, and more.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/legacy-nav-menu-block/`, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Use the block inserter to add the "Legacy Nav Menu" block to any post or page.
